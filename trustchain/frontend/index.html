On-Chain Proof

Proof NFT

Each issued proof mints a dedicated Algorand Standard Asset (ASA) representing ownership evidence.

How It Works

  1. Upload your file and issue a proof transaction.
  2. TrustChain hashes the file and records the hash on TestNet.
  3. TrustChain mints a one-of-one ASA linked to that proof.
  4. Anyone can verify the same file against the stored transaction.
